# Investment Details Extractor using open AI





## Setup 

1. **Clone the repo**

```bash
  https://github.com/alpha-titan/saturnai_frontend.git
```

2. **install dependencies**

```bash
  npm i
```
3. **Run the server**

```bash
  npm run dev
```
